# Image Grid Layout

## Descripción

Este es un proyecto de **Roadmap.sh** que consiste en crear una galería de imágenes con un layout de grid responsivo. El proyecto utiliza CSS Grid para distribuir las imágenes de manera estética y dinámica.

## Características

- **Grid Layout Responsivo**: Adapta el número de columnas según el tamaño de la pantalla
- **Breakpoints**:
  - Desktop: 5 columnas
  - Tablet: 3 columnas (máx. 1000px)
  - Mobile: 1 columna (máx. 600px)
- **Imágenes Dinámicas**: Las imágenes se cargan desde la carpeta `assets/img/`
- **Scroll Automático**: Permite desplazarse verticalmente cuando el contenido excede el viewport en pantallas pequeñas

## Estructura del Proyecto

```
image-grid-layout/
├── index.html
├── styles.css
├── README.md
└── assets/
    └── img/
        ├── 1.jpg
        ├── 2.jpg
        ├── ... (hasta 10.jpg)
```

## Instrucciones de Uso

1. Abre `index.html` en tu navegador
2. Las imágenes se cargarán desde la carpeta `assets/img/`
3. Redimensiona la pantalla para ver cómo se adapta el layout
